Use case 1 - Zero day útok

Funkce LIVE RESPONSE umožňuje operátorům ReaQty na daném zařízení vzdáleně provádět klíčové bezpečnostní operace, jako jsou:

výpis běžících procesů (show processes), výpis služeb (show services), výpis síťových spojení (show connections), stažení nebo smazání jakéhokoli souboru (download/delete file "%path"), výpis načtených modulů daného procesu (get process:modules "%pid"), výpis balíků (get packages), detailní informace o daném balíku (get package %packageName).

Ikony vedle příkazů značí podporu OS daného příkazu.

Klikněte na křížek + a následně na show processes.
